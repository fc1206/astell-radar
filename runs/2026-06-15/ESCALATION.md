# Escalation — 2026-06-15

## NEW TIER 1: Sentra (sentra.app)
- Bi-temporal context graph ('organizational memory') syncing 200+ work tools (Slack, Gmail, Drive, HubSpot, Linear, GitHub, Notion, Asana); surfaces what's true/owed/meant via NL query + MCP/REST memory API with source attribution
- Why Tier 1: Astell's exact mechanics — cross-tool ingest, graph-not-snapshot, cite-not-hallucinate, open-loop surfacing — wrapped in 'memory API' vocabulary; vocabulary never overrides mechanics (cf. Coworker)
- Evidence: https://sentra.app

## NEW TIER 1: PipesHub (pipeshub.com)
- Open-source + commercial enterprise workplace AI: cross-SaaS search + agentic automation over Slack, Notion, GitHub, Salesforce, Drive, Jira; knowledge-graph retrieval with pinpointed citations; self-hostable, model-agnostic, SOC 2 Type II
- Why Tier 1: Cross-SaaS search + graph + cited answers + agents = Astell's core mechanics; 'own your infra' open-source angle is the wedge against Glean/Astell
- Evidence: https://pipeshub.com

## NEW TIER 1: Doti (doti.ai)
- DIY agentic enterprise-search platform that searches a team's tools and builds workflow agents; acquired by Salesforce (announced Nov 13 2025, closed Q4 FY2026) to power agentic search in Slack/Agentforce
- Why Tier 1: Direct-lane enterprise search; acquisition = 4th consolidation of the category into a suite (after Dashworks→HubSpot, Unleash→Zendesk, Akooda→Tulip)
- Evidence: https://www.salesforceben.com/salesforce-buys-agentic-search-company-doti-its-eighth-acquisition-this-year/
